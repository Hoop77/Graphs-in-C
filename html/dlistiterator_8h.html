With the DListIterator you can simply mimic a foreach loop. It iterates through each node in the list starting with the first node and ending on the discrete end node.

Attention
When the loop is over, the iterator stands on the discrete end node. This means: If you want to get the data of the current node afterwards, you must set the iterator to a valid node first (not the discrete begin or end nodes)!

Attention
